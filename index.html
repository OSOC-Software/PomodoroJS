What is "Pomodoro technique" ?

The Pomodoro technique is a method for to focus on a specific task for a short time period of thirty minutes. It's very easy:

  1. Select a specific task and add in the box. Then click in "Add task!". The task will be added in a below list.
  2. When you start the task, click in button aside the task called "Start" and timer will start and you must do it for twenty five minutes. You do this task during this time. Important: Only this, you don't other task or activities.
  3. At the end of the time, you must rest for five minutes, for relax and regain energy.
  4. If you couldn't finish the task, simply, You can repeat the Pomodoro technique and finished the task
  5. Otherwise, You can complete other task or activities with this technique
